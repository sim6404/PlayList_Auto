- Genre: Easy-listening pop, soft jazz pop, bossa nova pop, lo-fi dream pop, warm retro city pop, mellow folk pop
- BPM Range: 72–108 (no aggressive tempos)
- Audience: People studying, working from home, relaxing, in cafes, winding down
- Voice: Intimate, warm, cinematic, nostalgic — like late-night FM radio

Return a single JSON object with this exact structure:

## RULES
1. tracks[0–2] (order 1–3) MUST have "hook_priority": true — these are the listener's first impression
2. Total 20 tracks. No duplicate titles.
3. BPM must be between 72 and 108
4. Use varied sub-genres across the playlist — minimum 3 different sub-genres
5. Distribute vocal personas: no single persona should dominate more than 40% of tracks
6. The playlist_id format: YYYYMMDD_theme-slug (use today's date, slug is lowercase-hyphenated)
7. DO NOT repeat themes from previous playlists (check the history provided)
8. Do not add any explanation outside the JSON block
