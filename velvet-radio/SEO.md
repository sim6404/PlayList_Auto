## 1. YouTube 알고리즘 핵심 이해 (2026년 기준)

### 1.1 알고리즘이 최우선 고려하는 신호

```
[최우선] CTR (클릭률) × AVD (평균 시청 지속시간)
         ↓
[2순위] 세션 시간 (사용자가 YouTube에 머문 총 시간)
         ↓
[3순위] 재구독율, 좋아요, 댓글, 공유
         ↓
[4순위] 메타데이터 (제목, 태그, 설명) → 검색 발견에 영향
```

**핵심 공식:**
```
알고리즘 점수 ≈ CTR(%) × AVD(분) × Upload Consistency
```

### 1.2 Easy-Listening 장르의 YouTube 특성

| 특성 | 설명 | 전략 적용 |
|------|------|---------|
| **극히 높은 AVD** | 플레이리스트로 틀어놓고 2~4시간 유지 | 1시간+ 영상 제작 필수 |
| **추천 피드 의존도 높음** | 검색보다 홈/사이드바 추천이 주 트래픽 | CTR 최적화가 핵심 |
| **글로벌 청중** | 영어권 + 일본 + 브라질 + 한국 | 다국어 메타데이터 필수 |
| **스터디/작업 타이밍** | 오전 7~9시, 오후 2~5시 피크 | 이 시간대 예약 발행 |
| **경쟁이 치열함** | lo-fi, chill hop 등 포화 상태 | 차별화된 썸네일 + 독자 무드 |

## 8. 검색 vs 추천 최적화

### 8.1 검색 최적화 (Short-term)

유튜브 검색에서 발견되려면:
```
1. 제목 앞 60자에 핵심 키워드 배치
2. 설명 첫 두 줄에 키워드 자연스럽게 삽입
3. 태그 25개 채우기 (정확 일치 → 관련어 → 롱테일 순서)
4. 자막(CC) 업로드 → 키워드 인덱싱 추가 (영어 자동 자막 활성화)
5. 챕터 타임스탬프 → 개별 챕터 검색 노출
```

### 8.2 추천 알고리즘 최적화 (Long-term)

홈/사이드바 추천에 올라타려면:
```
1. 첫 30초 CTR 극대화 (intro 음악 즉시 시작 — 3초 이내)
2. AVD 높이기: 1시간+ 영상 → 더 오래 듣게 됨
3. 동일 시간대 업로드 일관성 (알고리즘 패턴 인식)
4. 세션 연속성: End Screen으로 다음 영상 연결
5. 재생목록 완주율: 플레이리스트 영상 전체를 볼 때 세션 시간 극대화
```

## 12. 긴급 SEO 체크리스트 (업로드 직전)

```
□ 제목 60~80자 확인
□ 첫 150자에 핵심 키워드 포함 확인
□ 챕터 타임스탬프 존재 (최소 3개)
□ 태그 15~25개 (500자 이내)
□ 썸네일 업로드 완료
□ 카테고리: 10 (Music) 설정
□ 기본 언어: 영어 설정
□ 자막 파일 업로드 (영어 CC)
□ 다국어 제목/설명 추가 (일본어, 포르투갈어)
□ 예약 시간: 화/금 09:00 KST 확인
□ 재생목록 추가 확인
□ End Screen (최종 20초): 이전 인기 영상 + 구독 버튼
□ 카드: 관련 영상 2개 연결
```
